Room Temperature Controlled Fan SystemThis project is an automated cooling system that monitors room temperature using a DHT22 sensor. When the temperature exceeds a predefined threshold ($30^\circ\text{C}$), a relay is triggered to turn on a fan. The real-time temperature and fan status (ON/OFF) are displayed on an I2C OLED screen.✨ FeaturesReal-time Monitoring: Continuous temperature tracking with the DHT22 sensor.Automated Control: Fan automatically toggles based on a $30^\circ\text{C}$ threshold.Visual Feedback: 128x64 OLED display shows current temperature and system state.Simulation Ready: Fully functional on the Wokwi simulator.🛠️ ComponentsMicrocontroller: Arduino Uno (or compatible)Sensor: DHT22 (Temperature & Humidity)Display: SSD1306 OLED (128x64 I2C)Actuator: 5V Relay ModuleOutput: DC Fan (represented by relay logic)🔌 Pinout ConnectionsComponentComponent PinArduino PinDHT22VCC5VDataPin 2GNDGNDRelayVCC5VINPin 3GNDGNDOLEDVCC5VGNDGNDSCLA5 (SCL)SDAA4 (SDA)📚 Required LibrariesTo run this project, ensure you have the following libraries installed in your environment.Wokwi libraries.txt:PlaintextDHT sensor library
